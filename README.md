# Selenium SauceDemo Automation Project

## 📌 Project Overview

This project is an end-to-end automation testing project for the SauceDemo web application using Selenium WebDriver with Java.

It covers core automation concepts like locators, user interactions, validations, and complete purchase flow.

---

## 🚀 Features Covered

* Login functionality automation
* Product selection and navigation
* Add to cart functionality
* Checkout process automation
* Form submission (user details)
* Order completion validation
* Logout functionality

---

## 🛠️ Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* ChromeDriver

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Snehitha-Mullapudi/selenium_saucedemo_automation.git
```

2. Open the project in Eclipse or any IDE

3. Install dependencies using Maven

4. Run the main class:

```
selenium_saucedemo_automation.java
```
